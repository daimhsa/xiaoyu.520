# xiaoyu.520 - 表白网页 💕

一个浪漫的表白网页，包含动画效果、音乐播放和交互功能。

## 功能特性

✨ **动画效果**
- 心形飘落动画
- 柔和的页面过渡效果
- 浮动的标题动画

🎵 **音乐播放**
- 一键播放/暂停功能
- 支持循环播放

💌 **交互功能**
- 点击按钮显示表白文字：**你是我的一生所爱**
- 点击时触发心形雨效果

🎨 **设计风格**
- 粉色系渐变背景
- 现代化的界面设计
- 响应式布局（支持移动设备）

## 文件说明

- `index.html` - 网页主体结构
- `style.css` - 样式和动画设计
- `script.js` - 交互功能和逻辑

## 使用方法

1. 在浏览器中打开 `index.html`
2. 点击 "点击我" 按钮查看表白文字
3. 点击 "🎵 播放音乐" 按钮播放背景音乐
4. 欣赏心形飘落的浪漫效果

## 自定义说明

### 修改文字
在 `index.html` 中修改：
```html
<p id="message" class="message hidden">你是我的一生所爱</p>
```

### 修改音乐
在 `index.html` 中修改音乐 URL：
```html
<source src="https://music.163.com/song/media/outer/url?id=1372407605.mp3" type="audio/mpeg">
```

### 修改颜色
在 `style.css` 中修改粉色相关的颜色值

## 部署到 GitHub Pages

1. 进入仓库 Settings
2. 向下滚动到 GitHub Pages 部分
3. 选择 `main` 分支作为源
4. 访问 `https://daimhsa.github.io/xiaoyu.520/` 查看
